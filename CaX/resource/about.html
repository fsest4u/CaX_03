Music X NEO

By NOVATRON Co., Ltd.

Description

The NOVATRON Music X NEO is a remote controller app for smart phone and tablet PC, which will give you an easy and convenient control for NOVATRON's cocktailAudio N,X-series products such as N15(D), X14, X35, X45, X50 and more.

The NOVATRON Music X NEO enables you to control and use most of cocktailAudio N,X series' features and functions remotely; MusicDB play and management, file management like copy, move, delete, import to MusicDB and export from MusicDB, make a playlist and its management, playing online music services such as TIDAL, Deezer, Napster, Qobuz, Airable and HighResAudio etc

Applicable Models

• NOVATRON cocktailAudio N,X-series such as N15(D), X14, X35, X45, X50 and more

Main feature

• Simple graphic designed UI

• Intuitive and well organized screen make easy select of Devices,Music Source, Setup and Main Menu like "Home", "Now Playing", "Play Queue".

• Convenient playback on Music DB by Queue system for Music Server and music files in the mobile device

• Well organized folder structure Browsing

• Virtual Remote Control enables you to control all functions perfectly remotely

• Now playing screen shows song information, cover art and control menu for shuffle function, volume, stop, pause, next, previous.
